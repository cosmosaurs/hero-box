# Hero in a Box

Quickly fill your scenes with NPCs: pick **tags** (species, gender, age, profession, and more), then create actors with **portraits**, **tokens**, and **generated names** that fit your filters. 🦸

## How it works

Hero in a Box is the **generator**: it indexes tagged images from your chosen journals and compendiums, rolls a matching look, and assembles names from data you attach to journal pages. You control **which sources** to use in module settings.

## Content packs

The **core module** does **not** ship with portraits, predefined tag sets, or name lists. Pair it with **companion modules** (ours or the community’s) that provide compendiums or journals in the expected format.

Pack and extension list: **[Hero in a Box — Content Packs](https://github.com/cosmosaurs/hero-box/wiki/Content-Packs)**

## Installation

Install and enable **Hero in a Box** from the Foundry VTT module browser, or use the manifest URL from your release page. After enabling, open **Module Settings** to add data sources (journals/compendiums). Install at least one **[content pack](https://github.com/cosmosaurs/hero-box/wiki/Content-Packs)** module if you want ready-made tags, images, and names.
